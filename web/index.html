🔑 จัดการ API Key

ใส่ API Key อย่างน้อย 1 ตัวเพื่อเริ่มใช้งาน แนะนำเริ่มจาก Groq (ฟรี + เร็วที่สุด)

🤖 เชื่อมต่อกับ OpenClaw

ขั้นตอนง่ายๆ 3 ขั้นตอน:

  1. ใส่ API Key อย่างน้อย 1 ตัว (หน้า จัดการ API Key)
  2. เลือกผู้ให้บริการและโมเดลด้านล่าง
  3. คัดลอก config ไปวางใน OpenClaw แล้วรีสตาร์ท

เลือกผู้ให้บริการและโมเดล

📋 Config สำหรับ OpenClaw

🔧 วิธีตั้งค่าใน OpenClaw

# วิธีที่ 1: ใช้คำสั่ง (แนะนำ) openclaw config # วิธีที่ 2: แก้ไขไฟล์โดยตรง nano ~/.openclaw/openclaw.json # หลังจากแก้ไข config แล้ว: openclaw restart

🐳 Docker (แนะนำ)
💻 Manual

🐳 ติดตั้งด้วย Docker (ง่ายที่สุด)

ต้องมี Docker ติดตั้งก่อน

# 1. ดาวน์โหลด git clone https://github.com/tenotony/RouterAI.git cd RouterAI # 2. รันเลย! docker compose up -d # 3. เปิด Dashboard # http://localhost:8899 # อัพเดตเมื่อมีโค้ดใหม่: bash scripts/update.sh

💻 ติดตั้งด้วยมือ (ไม่ต้องใช้ Docker)

# 1. ดาวน์โหลด git clone https://github.com/tenotony/RouterAI.git cd RouterAI # 2. สร้าง Virtual Environment python3 -m venv venv source venv/bin/activate # 3. ติดตั้ง Dependencies pip install -r requirements.txt # 4. รัน Proxy (Terminal 1) python src/proxy.py # 5. รัน Dashboard (Terminal 2) python src/dashboard.py # 6. เปิด Dashboard # http://localhost:8899

⚡ One-Line Install

curl -fsSL https://raw.githubusercontent.com/tenotony/RouterAI/main/scripts/install.sh | bash
