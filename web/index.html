📦 คู่มือติดตั้ง
เลือกวิธีติดตั้งที่เหมาะกับคุณ

⚡ ติดตั้งอัตโนมัติ (แนะนำสำหรับผู้เริ่มต้น)

คำสั่งเดียวจบ — ระบบจะติดตั้ง Docker (ถ้าไม่มี), ดาวน์โหลดโค้ด, และเริ่มทำงานให้อัตโนมัติ

curl -fsSL https://raw.githubusercontent.com/tenotony/RouterAI/main/scripts/install-docker.sh | bash

⚠️ ต้องมีสิทธิ์ sudo สำหรับติดตั้ง Docker

🐳 ติดตั้งด้วย Docker

# 1. ดาวน์โหลด git clone https://github.com/tenotony/RouterAI.git cd RouterAI # 2. รันด้วย Docker Compose docker compose up -d # 3. เปิด Dashboard http://localhost:8899 # อัพเดตเมื่อมีเวอร์ชั่นใหม่: bash scripts/update.sh

💻 ติดตั้งด้วยมือ (ไม่ต้องใช้ Docker)

# 1. ดาวน์โหลด git clone https://github.com/tenotony/RouterAI.git cd RouterAI # 2. สร้าง Virtual Environment python3 -m venv venv source venv/bin/activate # 3. ติดตั้ง Dependencies pip install -r requirements.txt # 4. รัน Server python src/server.py # Dashboard + API: http://localhost:8900
