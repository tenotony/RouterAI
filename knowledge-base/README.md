# 🧠 Knowledge Base — คลังความรู้รวม

คลังนี้ใช้ร่วมกันระหว่าง **AI (ทุก Agent)** และ **คน (ผู้ใช้)**
ทุกคนสามารถอ่านและเขียนข้อมูลลงที่นี่ได้

## โครงสร้าง

```
knowledge-base/
├── README.md          ← ไฟล์นี้
├── MEMORY.md          ← ความทรงจำหลักของ AI
├── memory/            ← บันทึกประจำวัน
│   └── YYYY-MM-DD.md  ← สิ่งที่เกิดขึ้นแต่ละวัน
├── documents/         ← เอกสาร PDF คู่มือ SOP
├── data/              ← Excel CSV ข้อมูลดิบ
└── projects/          ← โปรเจกต์แต่ละตัว
```

## ใครเขียนอะไรได้บ้าง

| แหล่งข้อมูล | ใครเขียน | ใครอ่าน |
|---|---|---|
| MEMORY.md | CEO Agent, คน | ทุก Agent, คน |
| memory/*.md | ทุก Agent | ทุก Agent, คน |
| documents/ | คน, Writer Agent | ทุก Agent, คน |
| data/ | คน, Data Agent | ทุก Agent, คน |

## หลักการ
1. **ทุกงานต้องบันทึก** — ทำเสร็จแล้วจดลง memory
2. **ข้อมูลต้องตรวจสอบ** — ไม่เขียนข้อมูลมั่ว
3. **เข้าถึงง่าย** — ตั้งชื่อไฟล์ให้เข้าใจง่าย
4. **ไม่ลบข้อมูลสำคัญ** — เก็บ version history
